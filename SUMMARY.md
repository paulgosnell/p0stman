## Objective

Public-facing portfolio showcasing AI capabilities through interactive voice agents, case studies, and demos. Includes internal admin dashboard for business operations (CRM, invoicing, contracts, analytics).

## Tech Stack

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, Framer Motion
- **Backend:** Supabase (PostgreSQL + Auth), Vercel Serverless
- **Voice AI:** Google Gemini Live API (WebSocket streaming)
- **Icons:** Lucide React

## Key Features

1. **Voice Agent** - Real-time Gemini-powered conversational AI
2. **Case Studies** - 15+ luxury portfolio showcases
3. **Admin Dashboard** - CRM, invoicing, contracts, analytics
4. **Internationalization** - 6 languages supported
5. **Lead Tracking** - UTM parsing, session analytics, conversion metrics
